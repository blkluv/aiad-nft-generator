# AI NFT GENERATOR

A Next.js + Ethereum DApp That Generates Images Using OpenAI, Can Upload The Image To IPFS and Mints It to the Blockchain (Currently Working On Polygon, Arbitrum, Optimism, Goerli Testnet, Sepolia Testnet) ⚡
